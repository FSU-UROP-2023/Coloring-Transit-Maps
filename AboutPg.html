ABOUT

Subways maps have been considered to be a useful tool in navigating transportation in the everyday life of a commuter. However, there seems to be a deficiency in current subways maps not taking into account color blind individuals, making it inaccessible for them to easily read subway systems. There has been extensive research into improving the legibility of transit maps by using techniques like altering the geography (reword this https://onlinelibrary.wiley.com/doi/full/10.1111/cgf.14030) however, development on algorithmically selecting the optimal colors can be neglected. We aim to produce a program that can compute several dots on a graph that are as different as possible from one another, making them easier to read by the consumer. For this particular research topic, we use Colorspacious, a python library, to code a series of colors that are as far apart in the CIECAM02-UCS color space as possible, while taking into account different types of color blindness.We expect this research approach to improve the quality of virtual subways maps, making them easier to read and more accessible to use for individuals with color vision deficiency..
